# field_staff_app

Mobile app for field staff conducting in-home visits. Features:
- **Today's assignments** (list of visits, member info, directions)
- **Visit recording** (start/end, notes, intake forms, photos)
- **Document capture** (camera + upload to `visit_ingest_svc`)
- **Follow-up flagging** (mark items for care coordinator review)
- **Slack channel** (`#field-ops`) for AI-assisted questions

**Tech:** React Native or Flutter, camera integration, offline-first with sync.

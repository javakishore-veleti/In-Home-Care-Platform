# care_admin_portal

React web app for admin staff. Features:
- **Dashboard** (today's visits, open appointments, staff availability)
- **Staff management** (rosters, service areas, certifications)
- **Scheduling rules** (availability windows, travel-time buffers)
- **Member search** (lookup by ID, name, service area)
- **Visit review** (approve drafted summaries from `document_author_flow`)
- **Slack channel** (`#care-admin`) for AI-assisted operational questions

**Tech:** React, TypeScript, Vite, Tailwind CSS.

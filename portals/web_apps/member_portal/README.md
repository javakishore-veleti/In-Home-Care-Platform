# member_portal

React web app for members (patients). Features:
- **Sign up / sign in** (calls `auth_svc` via the API gateway)
- **Book appointment** (service type, preferred date/time, service area)
- **View upcoming visits** (status, assigned staff, time)
- **View past visits** (visit notes, follow-up items)
- **Profile management** (contact info, insurance, preferences)

**Tech:** React, TypeScript, Vite, Tailwind CSS.

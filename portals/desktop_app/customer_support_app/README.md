# customer_support_app

Desktop app for customer support agents handling inbound calls and chats. Features:
- **Member lookup** (search by ID, phone, name)
- **Visit history** (timeline view of all visits for a member)
- **Case creation** (escalation, dispute, complaint)
- **Appointment booking on behalf of member**
- **Slack channel** (`#customer-support`) for AI-assisted case resolution

**Tech:** Electron or Tauri, React frontend, calls API gateway.
